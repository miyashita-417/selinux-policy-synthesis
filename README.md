\# 最小特権を考慮したSELinuxポリシーの制約に基づく自動合成



電子情報通信学会 インターネットアーキテクチャ研究会 (IA) にて発表の論文「最小特権を考慮したSELinuxポリシーの制約に基づく自動合成」における検証用コードおよび補足資料のリポジトリです。



本研究では、運用意図（機能要件）を論理制約（SMT）へ変換し、Z3ソルバを用いて最小特権のSELinuxポリシーを自動合成する手法を提案しています。



\## リポジトリの構成

\- \*\*src/\*\*: 提案手法の実装および検証用プログラムのソースコード

&nbsp; - `target/mydaemon.c`: 検証に使用した仮想デーモンプロセス

&nbsp; - `tool/solver.py`: SMTソルバを用いたポリシー自動合成ツール

\- \*\*documents/\*\*: 論文紙面の都合上省略された機能要件および論理制約の詳細定義



\## 動作要件

\- Python 3.x

\- z3-solver
